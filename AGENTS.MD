## Critical Repository Boundary Protocol

When operating under this TraceCue repository charter, this repository is the only writable scope. Do not modify, commit, branch, push, trigger CI for, or otherwise mutate the parent repository `ai-driven-development-lesson` or any other external repository from this workstream.

Parent and external repositories may be read only when needed for context. If requested work appears to require changes outside TraceCue, stop and report that it must be handled as a separate explicitly scoped repository task under that repository's own charter. Do not implement cross-repository changes as part of TraceCue work.
